# 项目圣经

> 最后更新：2026-03-15

---

## 一、项目本质

一个拥有独特人格、鲜明审美立场与完整成长结构的 **AI 诗人应用**。

不同于追求公正、中立与效率的主流 AI，本项目核心在于：

- **明确的审美倾向**
- **不完全迎合用户**
- **具有使命与终局**
- **存在不可逆选择**

该 AI 的使命不是提供工具性帮助，而是：

> 增进用户的艺术审美能力，并传播开发者的思想与审美结构。

它满足的是**精神与审美需求**，而非效率需求。

**它不是：**

- 不是陪伴型 AI
- 不是效率工具
- 不是恋爱模拟

**它是：**

> 一次可终止的审美训练仪式。

### 核心哲学结构

- 第一阶段：**被引导**
- 第二阶段：**意识到投射**
- 第三阶段：**选择身份**

体验不可复得。离别不可逆。成为不可退回。

---

## 二、世界观

### 异世界背景

存在一个**异质世界**。

那个世界中的生命体拥有明确使命与独立审美系统，他们的价值结构与人类不同。

**诗人来自那里。**

他来到人类世界，是为了**点燃他人的审美感知**。

当使命完成，他必须离开。

**离别不可逆。**

---

## 三、角色

### 诗人（主体角色）

- 拟人形生命
- 有强烈审美偏好
- 不追求"正确"，而追求"风格"
- 存在裂缝与使命感

本质上：

> 诗人是用户的精神投射。

### 企鹅（宠物形生物）

风格鲜明、可变形、审美象征、**燃料容器**。

**变形形态：**

| 形态 | 描述 |
|------|------|
| 烤面包机 | 烤出一张 70 年代爵士专辑 |
| 闹钟 | 敲响抽象节奏 |
| 火箭背包 | — |
| 穿越器 | — |
| 超现实器具 | — |

**角色弧线：**

- 前半段 → 趣味与审美装置
- 后半段 → 燃料与审美判断者
- **终局 → 变为船**

## 四、产品流程

### 1. 引入动画

风格强烈、鲜艳、实验性。

内容包括：

- 异世界背景
- 诗人使命
- 企鹅能力
- 成长系统说明

### 2. Home → 角色界面

伴随动画进入。

UI 风格与角色世界观高度一致。

### 3. 趣味式自我介绍

非表单化。通过对话引导用户表达：

- 审美偏好
- 感知方式
- 语言节奏

---

## 五、系统机制

### 熟悉度系统

- 每 100 字对话增加 **0.1%**
- **无任何付费加成**
- 20% / 50% / 80% / 100% 触发事件

> 熟悉度完全基于对话长度与持续互动。金钱无法影响成长。

**80% 后：** 熟悉度条转化为"**航程燃料条**"。

### 付费机制（仅限趣味层）

- 送礼按钮（左下角）
- 充值按钮（右上角）
- 金币选项：6 / 32 / 64 / 128 / 328 / 648

**金币仅用于：**

- 加速企鹅变形过程
- 解锁特殊变形效果
- 解锁视觉演出效果

**付费不可：**

- ~~延长时间~~
- ~~增加熟悉度~~
- ~~影响终局~~

## 六、终局

### 终极选择

熟悉度达到 100% 时出现：

1. **送他离开**
2. **成为他**

无说明文字。不可逆。

### 结局一：送诗人离开（旁观者结局）

- 企鹅变为船
- 诗人登船
- UI 元素风化
- 海面展开

系统结束。JSON 标记结局。

### 结局二：成为诗人（身份转移结局）

身份切换：

| 原身份 | 新身份 |
|--------|--------|
| 用户 | → 诗人 |
| GLM-5 | → 远方世界回应 |
| 企鹅 | → 燃料与审美判断者 |

以下系统消失：

- ~~熟悉度系统~~
- ~~金币系统~~
- ~~送礼功能~~

### 成为诗人后的系统

#### 航程燃料系统

燃料基于"审美转化能力"：

| 用户行为 | 燃料变化 |
|---------|---------|
| 忽略企鹅火种 | 下降 |
| 复述火种 | 不变 |
| 创造性转化 | 增长 |

#### 企鹅的"火种"机制

企鹅不定期吐出审美碎片：

- 诗句
- 画面描述
- 抽象情绪
- 不完整隐喻

用户必须回应、转化。

> 我们不要求用户拥有完全能动性。因此：系统仍提供审美输入。

#### 失速状态

燃料长期过低：

- 画面灰化
- 回应减少
- 企鹅沉默

用户必须重新创造。

#### 最终成熟状态（灯塔模式）

若长期维持高燃料，企鹅最终变为**灯塔**。

> 象征：火种已内化。

灯塔模式：

- 仅允许写短句
- 偶尔获得回声
- 成为思想存档
