## 核心配色

### 主色（页面背景）
- **色名**：深空靛青
- **HEX 范围**：#0B1426 → #162447
- **用途**：所有页面背景、主容器底色

### 辅色（交互元素）
- **色名**：翡翠流光绿
- **HEX 范围**：#00E5A0 → #00C9A7
- **用途**：按钮、链接、进度条

### 强调色（通知/事件）
- **色名**：琥珀金
- **HEX 范围**：#FFB347 → #FF8C42
- **用途**：通知、事件触发、里程碑

### 点缀色（警告/离别）
- **色名**：深樱红
- **HEX 范围**：#C73E5C → #E84855
- **用途**：警告、离别内容

### 文字色
- **主要文字**：#E0E6ED
- **次要文字**：#8892A0

| 阶段 | 主导色彩 | 氛围 |
|------|---------|------|
| 0-19% 初遇 | （如：冷色调，翡翠绿+深靛青） | |
| 20-49% 相识 | （如：琥珀金开始渗入） | |
